Website Peternakan Ayam Petelur | Website

Laravel
Bootstrap
Chart.Js

Website Management dan Monitoring Ayam Petelur (MMAP) merupakan website yang digunakan untuk memanagement dan memonitoring telur pada kandang ayam. Website tersebut memiliki fitur review, grafik dan penilaian yang bisa deberikan untuk mengembangkan kembali fitur yang dirasa kurang. Selain itu website tersebut masih menggunakan data dummy sebagi acuan untuk menampilkan data pada End User. Website tersebut dibuat untuk tujuan tugas akhir kelompok yang dibangun menggunakan SDLC, dokumentasi dan menggunakan Work Breakdown Structure (WBS) sebagai acuan pengerjaan.

Website Hemo-Checker | IoT

Laravel
Bootstrap
Mysql
C++

Website Hemo-Checker merupakan website untuk memantau, memonitoring, dan merekapitulasi hasil aktivitas setelah melakukan pengukuran kadar hemoglobin pasien melalui alat yang diberi nama hemo-check. Website tersebut juga sudah terintegrasi dengan alat hemo-check secara real time. Website ini dibangun menggunakan framework laravel sebagai server side, bootstrap sebagai framework UI, mysql sebagai database management system dan C++ untuk menjembatani antar website dengan perangkat IoT melalui protokol http.

Website MOV-FI (Movie-Film)

React JS
Tailwind CSS
API TMDB
Firebase

Website MOV-FI merupakan website tempat untuk mengetahui informasi mulai dari rating film, pendapatan hingga film yang akan tayang. informasi website didapat dari pemanggilan API yang disediakan oleh TMDB (The Movie Database) untuk membantu para developer mengembangkan sebuah website. Website ini menggunakan React JS supaya lebih responsive, dinamis serta fetching API dengan sederhana.Selain itu Tailwind CSS untuk tampilan User Interface dari website tersebut, dan informasi API didapat dari TMDB yang bersifat open source. Untuk penyimpanan data user menggunakan Firebase realtime database. Website ini masih dilakukan pengembangan sampai tahap ini, untuk mengetahui lebih jauh mengenai proses pengembangnya dapat mengklik link demo dibawah ini.
